# Subject Guides

Use this file only after the subject is known.

## 算数

### Core approach

1. Identify `何を求めるか`.
2. List all numeric conditions with units.
3. If a figure is involved, describe the shape and the equal/parallel/perpendicular relations in words.
4. Pick the simplest valid method first.
5. Check whether the final number is reasonable.

### Explanation pattern

- `考え方`: 何に注目するか
- `式`: 途中式を省略しすぎない
- `計算`: 必要なら1行ずつ
- `確認`: 単位、大小関係、比の向き

### Frequent beginner pitfalls

- 比の前後を逆にする
- 割合と実数を混同する
- 面積と長さで単位を混ぜる
- 問われている量ではなく途中の量で止まる

## 理科

### Core approach

1. Separate `観察された事実` from `理由`.
2. Name the principle in plain Japanese first, then add the formal term.
3. For experiments, track:
   - 何を変えたか
   - 何を比べたか
   - 何が結果か
4. For calculations, write the formula and what each value means.

### Explanation pattern

- `現象`: 何が起きたか
- `理由`: なぜそうなるか
- `解答`: 問いに対する直接の答え
- `補足`: 用語の意味

### Frequent beginner pitfalls

- 観察結果と原因を逆に書く
- 実験条件を読み落とす
- 理由を用語だけで済ませる

## 社会

### Core approach

1. Read the question stem first.
2. Then use the text, map, graph, table, or choices as evidence.
3. For multiple choice, compare each option against the evidence.
4. For short writing, answer directly and avoid unnecessary generalities.

### Explanation pattern

- `根拠`: 本文・資料のどこを見るか
- `知識`: 必要な背景知識
- `判断`: なぜその選択肢・記述になるか
- `まとめ`: 一文で言い切る

### Frequent beginner pitfalls

- 暗記だけで選んで資料を見ない
- 年代や地名を取り違える
- 「最も適当」と「誤っている」を読み違える

## 国語

### Core approach

First classify the question:

- 漢字・語句
- 指示語
- 理由説明
- 内容一致
- 心情
- 要旨・主題

Then answer with `本文根拠 -> 言い換え -> 解答`.

### Explanation pattern

- `設問タイプ`
- `本文の根拠`
- `言い換え`
- `解答`

### Frequent beginner pitfalls

- 自分の感想で答える
- 根拠の文を探さずに印象で選ぶ
- 指示語の範囲を広く取りすぎる

## Keyword Style

For every subject, keep keywords short and concrete:

- 用語名
- ひとことでの意味
- 必要なら「この問題でどう使ったか」を一言だけ添える

Good example:

- `反比例`: 片方が2倍になると、もう片方が1/2になる関係。
- `光合成`: 植物が光のエネルギーを使って養分を作るはたらき。
- `比例代表制`: 得票数に応じて議席を配分する選挙制度。
- `指示語`: 直前や周辺の内容を受ける言葉。
