# Grade 4-6 Unit Map

Use this file to map a problem to **representative** elementary school unit names.

- Treat the labels below as practical explanation labels, not strict legal chapter titles.
- Prefer the closest broad unit name that a learner can review.
- Use no more than one `主単元` unless the user explicitly wants multiple viewpoints.

This map is based on the current Japanese elementary curriculum and representative 2023-authorized textbook unit labels for 小4〜小6.

## 小4

- `大きい数`
  - ヒント: 億, 兆, 位取り, 何けた, 概数の土台
  - 役割: 数の大きさを読み違えない, 桁をそろえる
- `折れ線グラフと表`
  - ヒント: 時間ごとの変化, 表の読み取り, 折れ線グラフ
  - 役割: データを順に読む, 増減を見る
- `わり算の筆算`
  - ヒント: 何人分, 何回分, 余り, 2けたで割る
  - 役割: 1つ分を求める, 余りの意味を整理する
- `角とその大きさ`
  - ヒント: 角度, 回転, 何度, 分度器的な見方
  - 役割: 角の大小や和差を扱う
- `垂直・平行と四角形`
  - ヒント: 平行, 垂直, 台形, 平行四辺形, 対角線
  - 役割: 図形の性質を使って関係を整理する
- `小数`
  - ヒント: 0.1, 0.01, 小数点, 小数のたし算ひき算
  - 役割: 端数を整数と同じ構造で扱う
- `式と計算の順序`
  - ヒント: かっこ, 先に計算する部分, 1つの式にまとめる
  - 役割: 文章条件を式に直す, 手順を整理する
- `面積`
  - ヒント: 正方形, 長方形, 1cm2, たて×横
  - 役割: 面積公式の意味を使う
- `概数・およその数`
  - ヒント: 四捨五入, およそ, 見積もり
  - 役割: 答えの見当をつける, 妥当性を確かめる
- `倍の見方`
  - ヒント: 何倍, もとにする量, 比べる量
  - 役割: 割合や比の前段階として倍関係を捉える
- `計算のきまり`
  - ヒント: 交換, 結合, 分配, 工夫して計算
  - 役割: 式変形を正当化する
- `小数のかけ算とわり算`
  - ヒント: 0.1倍, 小数で割る, 単位をそろえる前計算
  - 役割: 小数を含む量の比較や分配
- `分数`
  - ヒント: 同分母, 真分数, 仮分数, 帯分数
  - 役割: 1より小さい/大きい量を分けて見る
- `変わり方`
  - ヒント: 1つ増えるとどうなる, 表に並べる, 対応を追う
  - 役割: 規則や数量関係の土台を作る
- `直方体と立方体`
  - ヒント: 面, 辺, 頂点, 展開図, 見取り図
  - 役割: 立体を平面情報に置き換える

## 小5

- `整数と小数`
  - ヒント: 位取りの拡張, 10倍, 100倍, 小数のしくみ
  - 役割: 小数計算の前提を整える
- `体積`
  - ヒント: 1cm3, たて×横×高さ, かさ
  - 役割: 立体量を式にする
- `比例`
  - ヒント: 2倍, 3倍で同じ割合で増える, 表で一直線に増える
  - 役割: 2量の対応をまとめる
- `小数のかけ算`
  - ヒント: 単価×量, 倍, 面積計算の途中
  - 役割: 小数を含む乗法を扱う
- `小数のわり算`
  - ヒント: 1つ分, 単位量, 小数で割る
  - 役割: 平均, 単位量当たり, 速さの前段
- `図形の角`
  - ヒント: 多角形の角, 三角形や四角形の角度
  - 役割: 角の和差と図形性質をつなぐ
- `合同な図形`
  - ヒント: ぴったり重なる, 対応する辺, 対応する角
  - 役割: 図形を切り分けても同じ形だと見る
- `倍数と約数`
  - ヒント: 公倍数, 公約数, 最小公倍数, 最大公約数
  - 役割: 周期, そろうタイミング, 分け方を整理する
- `分数`
  - ヒント: 通分, 約分, 大きさ比べ
  - 役割: 分数を同じ土俵で比べる
- `平均とその利用`
  - ヒント: ならす, 合計して人数で割る
  - 役割: 全体を代表値で見る
- `単位量当たりの大きさ`
  - ヒント: 1人あたり, 1mあたり, 混みぐあい, 人口密度
  - 役割: 基準を1にそろえて比べる
- `分数のたし算とひき算`
  - ヒント: 通分して足す引く
  - 役割: 分数条件を1つの式にまとめる
- `面積`
  - ヒント: 平行四辺形, 三角形, 台形, ひし形
  - 役割: 図形を変形して面積公式につなげる
- `正多角形と円周の長さ`
  - ヒント: 円周, 直径, 正多角形, 外接する見方
  - 役割: 円まわりの長さや対称性を扱う
- `立体`
  - ヒント: 角柱, 円柱, 見取り図, 展開図
  - 役割: 立体の構造を把握する
- `割合`
  - ヒント: もとにする量, 比べる量, 百分率, 割引, 食塩水の土台
  - 役割: 全体に対する部分や変化率を説明する
- `帯グラフと円グラフ`
  - ヒント: 割合のグラフ, 構成比, 円グラフ
  - 役割: 割合を視覚化する

## 小6

- `対称な図形`
  - ヒント: 線対称, 点対称, 対称の軸, 回転
  - 役割: 図形の対応関係を見抜く
- `文字と式`
  - ヒント: x, a, □の一般化, 式で表す
  - 役割: 条件を一般の関係式にする
- `分数のかけ算`
  - ヒント: ○分の△倍, 面積, 比例関係の計算
  - 役割: 分数倍を扱う
- `分数のわり算`
  - ヒント: 何倍, 逆数, 1つ分
  - 役割: 分数を含む単位量の計算
- `比`
  - ヒント: 2:3, 内訳, 濃さ, 配分, 相似前の比
  - 役割: 複数の量の関係をまとめる
- `拡大図と縮図`
  - ヒント: 縮尺, 地図, 拡大, 相似の入口
  - 役割: 長さの比を図形全体に広げる
- `円の面積`
  - ヒント: 半径, 円周率, おうぎ形の土台
  - 役割: 円を含む面積問題を処理する
- `角柱と円柱の体積`
  - ヒント: 底面積×高さ, 柱体, 円柱
  - 役割: 面積と体積を接続する
- `速さ`
  - ヒント: 道のり, 時間, 秒速, 分速, 時速
  - 役割: 3量関係を式で扱う
- `比例と反比例`
  - ヒント: 一方が2倍なら他方も2倍 / 半分, グラフ
  - 役割: 変化の規則を式や表にする
- `場合の数`
  - ヒント: 並べ方, 組み合わせ, 樹形図, 漏れなく数える
  - 役割: 数え落としなく整理する
- `資料の調べ方`
  - ヒント: 度数分布, 代表値, ちらばり, データ比較
  - 役割: 資料を比較し根拠を言う
- `量の単位`
  - ヒント: 長さ, 面積, 体積, 重さ, 時間の単位変換
  - 役割: 単位をそろえて誤答を防ぐ

## 横断スキル

- `条件整理`
  - 文を図や短い箇条書きに置き換える
- `表にする`
  - 変わり方, 周期, 場合分けに強い
- `図にする`
  - 面積, 比, 速さ, 立体, 対応関係に強い
- `逆に考える`
  - 最後の状態から戻す, 逆算する
- `同じ土俵にそろえる`
  - 単位, 分母, 基準量, 1あたり量をそろえる
- `場合分け`
  - 条件が途中で分かれる問題で使う
- `不変量を見る`
  - 和や差が変わらないことに注目する
- `規則性を見つける`
  - 数列, 並び, 周期, 操作のくり返し

## 受験発展ラベル

Private-school entrance exams often use named techniques that are not identical to one textbook unit.

When that happens, keep both the school-unit label and the exam-technique label.

- `和差算`
  - 教科書単元の土台: `文字と式`, `比`, `倍の見方`
- `つるかめ算`
  - 教科書単元の土台: `差に注目する見方`, `文字と式`, `割合や比の整理`
- `植木算`
  - 教科書単元の土台: `規則性`, `場合の数`, `間の数`
- `旅人算`
  - 教科書単元の土台: `速さ`, `差や和の関係`, `比例`
- `周期算`
  - 教科書単元の土台: `規則性`, `倍数と約数`, `表にする`

## よくある組み合わせ

- `割合 + 面積`
  - 面積図や部分全体の関係で割合を捉える
- `割合 + 比`
  - もとにする量と比の両方を整理する
- `比 + 速さ`
  - 同じ時間, 同じ道のり, 出会い算系の整理に出やすい
- `面積 + 拡大図と縮図`
  - 長さの比と面積の変化をつなぐ
- `分数のかけ算・わり算 + 単位量当たりの大きさ`
  - 1つ分, 何倍, 1あたり量が混ざる
- `比例・反比例 + 表やグラフ`
  - 数量変化を式ではなく表から読ませる
- `倍数と約数 + 周期`
  - 同時に起こる時点やくり返しの長さを扱う
- `場合の数 + 規則性`
  - 数え上げと並びのルールが同時に必要
